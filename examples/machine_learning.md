# Основы машинного обучения

Машинное обучение изучает методы, которые позволяют программе находить закономерности в данных и использовать их для прогнозов.

В supervised learning модель обучается на примерах, где для каждого объекта известен правильный ответ. Такие ответы называют метками. Классификация писем на спам и не спам является примером supervised learning.

В unsupervised learning модель получает данные без заранее известных меток. Она ищет скрытую структуру: кластеры, похожие группы объектов или сжатые представления.

Градиентный спуск обновляет параметры модели в направлении уменьшения функции потерь. На каждом шаге алгоритм вычисляет градиент и делает небольшой шаг с заданной скоростью обучения.
