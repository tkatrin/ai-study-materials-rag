# Нейронные сети

Нейронная сеть состоит из слоев, которые преобразуют входные данные в выходной результат. Каждый слой использует веса, смещения и функцию активации.

Обратное распространение ошибки вычисляет вклад каждого параметра в итоговую ошибку. После этого оптимизатор обновляет веса модели.

Переобучение возникает, когда модель слишком хорошо запоминает обучающие данные и хуже работает на новых примерах. Для борьбы с переобучением используют регуляризацию, dropout, увеличение данных и раннюю остановку.
